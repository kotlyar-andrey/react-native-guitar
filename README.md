# react-native-guitar

Library for playing guitar sounds (metronome, chords, beats, etc)

## Installation

```sh
npm install react-native-guitar
```

## Usage

```js
import { multiply } from 'react-native-guitar';

// ...

const result = await multiply(3, 7);
```
